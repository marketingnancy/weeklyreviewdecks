# Weekly Review Decks

Static dashboards / review decks for Hello Nancy marketing, published via **GitHub Pages**.

- **localization/** — Localization Daily dashboard (non-English markets). A static snapshot baked from the source dashboard; rebuild with `python3 -m localization.web.build_static` in the source repo and copy `build_static/` here.

Add new decks as sibling folders, then link them from the root `index.html`.
